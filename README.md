## Português - BR
Sistema de perguntas e resposta simples, feito em 5 horas, para um trabalho da faculdade. Este projeto não foi feito para ser publicado oficialmente.

Consiste em uma tela com uma pergunta e quatro alternativas, onde o grupo A ou B, que é decidido atravéz da forma que o orador desejar, tem a chance de responder. Se a resposta estiver correta, ponto para o grupo, se estiver errada o ponto vai para o outro grupo.

O sistema é bem simples, não há conexão com banco de dados, e as perguntas estão no código principal, que tem lugar para 10 perguntas, podendo ser extendido. 

Na tela principal nos temos três botões escondidos, dois logo abaixo do nome dos grupos, onde no clique é verificada a resposta e computado os pontos (primeiro é necessário selecionar a opção). E um terceiro botão escondido no topo da tela, para ir para a proxíma pergunta. Você deve estar se perguntando, por que escondidos? Simples, apresentei esse questionário para toda a turma e eles não precisaram ver os botões.

O aplicativo foi feito para Windows Universal Apps, e foi testado posteriormente em um Lumia 1520, porém o design não está preparado para celulares (visto que o projeto foi feito para rodar em um computador muito especifico: o meu).

Há muito o que melhorar, porém não vou dar continuidade a este projeto já que ele compriu com o seu dever.

## English - EN

A simple trivia game, done in 5 hours, for university work. This is not for release propouse.

It is a screen with one question and four ansewers, where the group A or B, that is decided in old ways (who screen lowter, or any ideias) has the chance to respond. If the answer is correct, then point to the group. If is not, point to the other group.

It is a simple sistem, no connection with database, and the questions is in the code themself, has spot for 10 questions, but you can use more with a little adaptation.

On the main screen (and the only), we have three hidden buttons, two bellow the group names, where on click it is checked the answer and computed the points (fist it is needed to select the option). And a third buttom on the top, also hidden, to go to the next question. You may be asking, why it is hidden? Well, I show this trivia in a screen for all the class, they don't needed to see this buttons and I haven't enought time to make two differents screens (one with this button and other with the questions).

The app was done for Windows Universal Apps, and it was tested later in a Lumia 1520, the user interface was not great, as espected, cause I did it to run only on my pc. But nothing that a little work fix it.

There is so much to improve, but I'm not going to continiu this, it is already completed your propouse.

By the way, the code (variable) is in portuguese, but nothing hard to understand. Good luck with no comments on the code!
